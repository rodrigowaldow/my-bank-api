# My Bank API
API integrada ao MongoDB cujo o schema dos dados será definido pelo mongoose. Esta API terá alguns endpoints para manipulação dos dados utilizando estruturação e persistência dos dados pelo mongoose.

## Tecnologias:
- JavaScript;
- NodeJS;
- Express;
- MongoDB;
- Swagger;
- Winston;
